01 File Structure

Extract the zip file to see the clean and organized folder structure:

PortfolioPro/
├── assets/
│   ├── css/
│   │   └── style.css       # Custom styles & animations
│   ├── js/
│   │   └── script.js      # Theme & Mobile Menu logic
│   └── images/            # All local assets
├── documentation/         # This help guide
│   ├──index.html/
│   └──img/
├── index.html             # Main template file
└── LICENSE.txt            # Usage rights

02 Editing Content

To change text, find the comments in index.html:

<!-- EDIT CONTENT HERE -->
<h1>Hi, I'm Your Name</h1>
<p>UX/UI Designer & Developer</p>

Figure 1: VS Code showing how to edit the content

03 Editing Images

Go to assets/images/ and replace the images with your own files using the same names, or update the paths in the HTML:

<img src="/assets/images/your-photo.jpg" alt="Description">

Figure 1: VS Code showing how to update image paths

04 Contact Form Setup

This template uses Formspree for the contact form. No PHP is required!

  • Step 1: Get your unique ID from Formspree.io.
  • Step 2: Find the form tag in index.html.
  • Step 3: Update the action attribute: 
<form action="https://formspree.io/f/YOUR_ID" method="POST">

Figure 1: VS Code showing how to update Contact Form

05 Privacy Policy & Terms of Service

To ensure your site is professional, we integrated legal modals. You don't need separate HTML files; everything is in index.html.

How to edit the legal text:

  1. Open index.html.
  2. Search for id="privacyModal" or id="termsModal".
  3. Change the placeholder text inside the <p> tags to match your personal or company information.

Figure 1: VS Code showing how to update Privacy Policy & Terms of Service
