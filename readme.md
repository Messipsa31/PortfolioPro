# 🚀 PortfolioPro | Premium Personal Portfolio Template

**PortfolioPro** is a modern, high-performance, and fully responsive portfolio template designed for creative professionals, developers, and designers. Built with **Tailwind CSS**, it features a sleek Dark Mode, smooth animations, and is fully optimized for SEO.

---

## 💎 Key Features

* **🌓 Smart Dark Mode:** Built-in dark/light theme toggle that remembers user preference via LocalStorage.
* **📱 Ultra Responsive:** Perfectly optimized for mobile, tablet, and desktop screens.
* **⚡ Optimized Performance:** Clean, modular code structure for lightning-fast loading speeds.
* **🔍 SEO & Social Ready:** Pre-configured with Meta Tags, Open Graph (OG), and JSON-LD Structured Data.
* **🎨 Modern UI/UX:** Features Glassmorphism effects, smooth scrolling, and professional typography.
* **📧 Functional Contact Form:** Ready to be integrated with **Formspree** without any backend coding.

## 🛠️ Quick Start Guide
1. Personalize Content
Open index.html and look for the comments labeled ``. Replace the placeholders with your name, profession, and bio.

2. Update Assets
Replace the placeholder images in assets/images/ with your own photos. Ensure you keep the same file names or update the paths in index.html.

3. Setup Contact Form
This template is pre-configured for Formspree:

Register for a free account at Formspree.io.

Create a new form and copy your unique Form ID.

In index.html, find the <form> tag and update the action attribute:
action="https://formspree.io/f/YOUR_ID"

4. Legal & Compliance
- Integrated Modals: Privacy Policy and Terms of Service are built-in as modern pop-up modals.
- Easy Customization: You can update the legal text directly inside the `index.html` file without creating extra pages.

5. Global Colors
To change the primary theme color, modify the tailwind.config section in the <head> of index.html:
```
theme: {
    extend: {
        colors: {
            primary: "#3B82F6", // Replace with your brand color code
        }
    }
}
```
